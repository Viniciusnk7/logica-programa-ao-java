📚 Sistema de Aprovação Escolar

Projeto desenvolvido em Java para praticar estrutura condicional, entrada de dados, cálculo de média e lógica de programação.

🚀 Funcionalidades

Cadastro do nome do aluno
Entrada das 4 notas bimestrais
Cálculo automático da média final

Verificação da situação do aluno:

✅ Aprovado
⚠️ Recuperação
❌ Reprovado

🛠️ Tecnologias utilizadas

Java
Eclipse IDE
Scanner
Locale

📋 Regras de aprovação

Média Final	Situação
70 ou mais	Aprovado
40 até 69.9	Recuperação
Abaixo de 40	Reprovado

💻 Exemplo de execução

Digite o nome do aluno:
Carlos

Digite a nota do 1° Bimestre:
80

Digite a nota do 2° Bimestre:
75

Digite a nota do 3° Bimestre:
60

Digite a nota do 4° Bimestre:
90

RESULTADOS

Aluno: Carlos
Média final: 76.2
APROVADO!
