💰 Simulador de Empréstimo Simples em Java

Projeto desenvolvido para praticar lógica de programação em Java.

O sistema simula um empréstimo simples, calculando:

valor solicitado
taxa de juros
valor total com juros
quantidade de parcelas
valor de cada parcela

🚀 Tecnologias utilizadas
Java
Eclipse IDE
Scanner
Locale

📚 Conceitos praticados
Entrada de dados
Saída de dados
Variáveis
Operações matemáticas
Cálculo de porcentagem
Organização de código
Lógica de programação

⚙️ Funcionalidades

✅ Cadastro do nome do cliente
✅ Cálculo de juros
✅ Cálculo do valor total
✅ Simulação de parcelamento
✅ Exibição completa do empréstimo

🖥️ Exemplo de execução

Digite o seu nome:
Vinicius

Digite o valor que voce deseja R$ :
1000

Porcentagem da taxa de juros (%):
10

Digite a quantidade das parcelas que vc deseja:
5

Cliente: Vinicius
Valor do emprestimo : R$ 1000.0
juros: R$ 100.0
total: R$ 1100.0
