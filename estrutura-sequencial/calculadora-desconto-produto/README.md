🧮 Calculadora de Desconto de Produto

Projeto simples desenvolvido em Java para praticar lógica de programação, entrada de dados, cálculos matemáticos e saída formatada no console.

📌 Funcionalidades

✅ Ler o nome do produto
✅ Ler o preço original
✅ Ler a porcentagem de desconto
✅ Calcular o valor descontado
✅ Mostrar o valor final do produto

💻 Tecnologias utilizadas
Java
Eclipse IDE
Programação Orientada à Lógica

📷 Exemplo de execução
Digite o nome do produto:
Notebook

Digite o preço original:
3500

Digite a porcentagem de desconto (%):
10

nome do produto: Notebook
valor original: R$ 3500.0
porcentagem de desconto: % 10.0
valor descontado: R$ 350.0
valor final: R$ 3150.0

🎯 Objetivo do projeto

Esse projeto foi criado para fortalecer conceitos de:

Variáveis
Entrada de dados com Scanner
Operações matemáticas
Porcentagem
Saída de dados no console
Estrutura sequencial em Java
